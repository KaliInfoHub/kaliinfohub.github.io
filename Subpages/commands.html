Kali Linux Command Reference

This guide is a collection of essential Kali Linux commands grouped by category. Whether you're navigating the system, gathering info, or launching tools — this reference has you covered.

1. System Information & User Management

whoami

Displays the currently logged-in user.

# Example: whoami
uname -a

Displays kernel and OS information.

# Example: uname -a
id

Shows user ID (UID), group ID (GID), and associated groups.

passwd

Change password of current (or specified) user.

2. Networking & Scanning

ip a

Shows IP addresses of network interfaces (modern replacement for ifconfig).

netdiscover

Finds live hosts on your local network via ARP requests.

nmap -sS -T4 192.168.1.0/24

Performs a fast stealth scan on an entire subnet.

ping -c 4 google.com

Sends 4 ICMP packets to test network reachability.

netstat -tulnp

Lists all listening ports and associated services.

3. File & Directory Management

ls -la

Lists files in long format, including hidden ones.

cd /etc

Changes the working directory to `/etc`.

cp file.txt /tmp/

Copies a file to `/tmp` directory.

rm -rf /tmp/testdir

Forcefully deletes a directory and its contents.

cat /etc/passwd

Displays contents of the system password file.

4. Privilege Escalation

sudo su

Switches to root user with full privileges.

sudo -l

Lists allowed (and forbidden) sudo commands for the current user.

find / -perm -4000 -type f 2>/dev/null

Searches for SUID binaries (common privilege escalation targets).

5. Package Management & System Updates

apt update && apt upgrade

Refreshes package index and upgrades installed software.

apt install nmap

Installs Nmap from the Kali package repository.

6. Common Pentesting Tools

msfconsole

Starts the Metasploit Framework console for launching exploits.

hydra -l admin -P rockyou.txt ssh://192.168.1.10

Uses Hydra to brute-force SSH login using a username and password list.

airmon-ng start wlan0

Enables monitor mode on a wireless adapter for packet capturing.

aircrack-ng capture.cap -w wordlist.txt

Cracks Wi-Fi handshake using a wordlist attack.

burpsuite

Launches Burp Suite GUI for web application testing.

nikto -h http://192.168.1.100

Scans a web server for known vulnerabilities and misconfigurations.
