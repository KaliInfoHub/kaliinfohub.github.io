What is Metasploit?

Metasploit is an open-source penetration testing platform that enables security professionals and ethical hackers to identify, exploit, and validate vulnerabilities in systems. It includes a comprehensive set of tools for reconnaissance, exploitation, privilege escalation, and post-exploitation.

Developed originally by H.D. Moore, Metasploit has grown into a vital tool in cybersecurity, integrated into nearly every professional penetration tester’s workflow.

Basic Usage

Start Metasploit:

msfconsole

Search for an exploit:

search vsftpd

Use an exploit module:

use exploit/unix/ftp/vsftpd_234_backdoor

Set options and run:

set RHOST 192.168.1.10
set RPORT 21
exploit

Get a list of payloads:

show payloads

Practical Example

Suppose you're testing a target machine running an outdated FTP service vulnerable to backdoor access. Using Metasploit, you can:

  • Scan the network for hosts using Nmap
  • Identify service versions
  • Launch an appropriate exploit
  • Gain a shell or meterpreter session
  • Perform post-exploitation actions like hash dumping or privilege escalation

Key Modules Explained

  • Exploit: The code that takes advantage of a vulnerability
  • Payload: What gets executed after the exploit is successful (e.g., reverse shell)
  • Auxiliary: Tools for scanning, fuzzing, sniffing, etc.
  • Post: Modules for post-exploitation, like gathering system information
  • Encoder: Obfuscates payloads to avoid detection

Tips for Beginners

  • Use info after selecting a module to learn what it does
  • Start with known vulnerable machines like Metasploitable or TryHackMe labs
  • Use setg to globally set options across modules
  • Save your sessions with sessions -s and list them using sessions -l
  • Check out official Metasploit docs
