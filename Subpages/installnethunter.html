Kali NetHunter is the official penetration testing platform for Android devices. It adds a powerful set of tools to your phone, making mobile hacking possible on the go.

Installation Requirements

  • An Android device (rooted) with an unlocked bootloader
  • Custom Recovery installed (TWRP is recommended)
  • Sufficient internal storage (at least 8GB free)
  • NetHunter image matching your device

Step-by-Step Installation

  1. Unlock Bootloader
    - Go to Settings → About Phone → tap Build Number 7 times to unlock Developer Options
    - Enable OEM Unlock and USB Debugging
    - Use fastboot oem unlock from your PC (may vary by brand)
    Warning: This will factory reset your device!
  2. Install TWRP Recovery
    - Download the correct TWRP image for your phone from twrp.me
    - Flash via fastboot: fastboot flash recovery twrp.img
    - Boot into recovery (usually Volume Up + Power)
  3. Download NetHunter Files
    - Visit kali.org/get-kali/#kali-mobile
    - Choose the correct build for your device (Generic ARM/ARM64 or specific)
    - Copy the zip to your phone’s storage
  4. Flash NetHunter via TWRP
    - In TWRP, select “Install” and navigate to the NetHunter zip
    - Swipe to confirm flash
    - Optional: Flash additional modules (e.g., NetHunter kernel if provided)
  5. First Boot + Setup
    - Reboot to system
    - Open the NetHunter app (if not present, install from NetHunter Store)
    - Grant root access and install core components
    - Setup completed!

Note: Not all devices are officially supported. Always back up your device before attempting modifications.
