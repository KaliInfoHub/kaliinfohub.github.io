This guide provides a full walkthrough to safely install Kali Linux alongside Windows, so you can select your OS at startup.

Step-by-Step Dual Boot Setup

  1. Back Up Important Files
    Before making any changes to your disk, back up all important files to an external drive or cloud storage.
  2. Download Kali Linux ISO
    Visit the official site and download the Kali Installer ISO:
    https://www.kali.org/get-kali/#kali-installer-images
  3. Create a Bootable USB
    Use a tool like Rufus on Windows:
    - Plug in a USB drive (at least 8GB)
    - Launch Rufus and select the Kali ISO
    - Set “Partition Scheme” to MBR (for BIOS) or GPT (for UEFI)
    - Click Start to write the ISO
  4. Shrink Windows Partition
    Free up space for Kali using Disk Management:
    - Press Win + X → select "Disk Management"
    - Right-click your Windows partition → choose “Shrink Volume”
    - Shrink by at least 20GB (recommended: 30–50GB)
    - Leave the new space unallocated
  5. Boot from USB
    - Restart your computer and press the BIOS/UEFI key (usually F2, Del, Esc, F12)
    - Select your USB as the boot device
  6. Begin Kali Installation
    - Choose “Graphical Install” on the boot menu
    - Select language, location, and keyboard settings
    - Configure hostname and create user account
  7. Partition Disks
    - Choose “Guided – use the largest continuous free space”
    - Select “All files in one partition”
    - Confirm changes and write to disk
    - This installs Kali on the space you freed up earlier
  8. Install GRUB Bootloader
    When asked about GRUB, choose “Yes” and select the boot drive (usually `/dev/sda`)
    GRUB allows you to pick Kali or Windows during startup
  9. Finish Installation
    - Remove the USB when prompted
    - Reboot and you’ll see the GRUB menu with both Kali and Windows!

You're Done! Welcome to dual-booting! Use Kali for hacking, forensics, or learning — while keeping Windows available when needed.
